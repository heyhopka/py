Блок 1 · Ввод и вывод

Функции input() и print()

input() используется, чтобы спросить что-то у пользователя (например, имя котика или его возраст). print() выводит информацию на экран — текст, числа, переменные.

  • input(подсказка) всегда возвращает строку (str).
  • Если нужно число (например, возраст кота) — оборачиваем в int() или float().
  • print() может выводить несколько значений сразу, через запятую.
  • Параметры sep и end управляют разделителем и концом строки.
Пример: спрашиваем имя кота и его возраст, потом выводим:
cat_name = input("Как зовут котика? ") cat_age = int(input("Сколько ему лет? ")) print("Это котик по имени", cat_name) print("Ему", cat_age, "лет")
Настройка разделителя и конца строки:
print("Кот", "Мурзик", sep=": ", end=" любит спать.\\n") # Выведет: Кот: Мурзик любит спать.
input() → всегда str int(input()) → число print(a, b, c)
Блок 2 · Классы и объекты

Класс Cat — описываем котика

Класс — это свой тип данных. В примерах ниже мы создаём класс Cat, который хранит имя кота, его возраст и любимый корм.

Определение класса Cat:
class Cat: def __init__(self, name, age, favorite_food): self.name = name # имя кота (строка) self.age = age # возраст (int) self.favorite_food = favorite_food # любимый корм (строка)
  • class Cat: — объявление класса.
  • def __init__(...) — конструктор, вызывается при создании кота.
  • self — ссылка на конкретного котика.
  • self.name, self.age, self.favorite_food — атрибуты (поля) объекта.
Создание объекта и доступ к его полям:
murzik = Cat("Мурзик", 3, "тунец") print(murzik.name) # Мурзик print(murzik.age) # 3 print(murzik.favorite_food) # тунец
__init__ — конструктор self.поле = значение cat = Cat(...)
Блок 3 · Списки и функции

Список котиков и функции для работы с ним

Вместо списка учеников теперь — список котов. Мы можем хранить сразу много объектов Cat, а функции будут работать с этим списком: выводить всех котиков, выводить только котят, считать средний возраст.

Создание списка и добавление котов:
cats = [] cat1 = Cat("Мурзик", 3, "тунец") cat2 = Cat("Снежок", 1, "курочка") cat3 = Cat("Луна", 5, "сухой корм") cats.append(cat1) cats.append(cat2) cats.append(cat3)
  • [] — пустой список.
  • append() — добавить элемент в конец списка.
  • for cat in cats: — перебор всех котиков.
  • len(cats) — количество котов в списке.
Функция для вывода всех котов:
def print_cats(cats): for cat in cats: print(cat.name, "-", cat.age, "лет,", "любит", cat.favorite_food) print()
Функция для вывода только котят (например, до 2 лет):
def print_kittens(cats): print("Котята (до 2 лет):") for cat in cats: if cat.age <= 2: print(cat.name) print()
Функция для подсчёта среднего возраста котов:
def average_cat_age(cats): total_age = 0 for cat in cats: total_age += cat.age average = total_age / len(cats) print("Средний возраст котиков:", average)

Блок 4 · Файлы

Чтение и запись файлов: cats.txt и cats_quotes.txt

Раньше у тебя были файлы pupils.txt и quotes.txt. Логика та же самая, только теперь это, например, cats.txt с информацией о котах и cats_quotes.txt с забавными “мыслями” котиков.

Чтение котов из файла cats.txt

Файл содержит строки формата: Имя Возраст ЛюбимыйКорм
cats = [] with open("cats.txt", "r", encoding="utf-8") as file: for line in file: data = line.split() # ["Мурзик", "3", "тунец"] name = data[0] age = int(data[1]) favorite_food = data[2] cat = Cat(name, age, favorite_food) cats.append(cat)
  • with open(..., "r") as file: — открыть файл для чтения.
  • encoding="utf-8" — чтобы нормально читать русский текст.
  • for line in file: — читать файл построчно.
  • line.split() — разделить строку по пробелам на части.

Чтение файла с “цитатами” котиков

with open("cats_quotes.txt", "r", encoding="utf-8") as file: for line in file: print(line)

Запись новой “цитаты” кота в файл

quote = input("Что бы сказал ваш кот? ") cat_name = input("Как зовут этого кота? ") with open("cats_quotes.txt", "a", encoding="utf-8") as file: file.write(quote + "\\n" + "(" + cat_name + ")" + "\\n")
  • "r" — чтение (read).
  • "w" — запись, стирает содержимое (write).
  • "a" — добавление в конец файла (append).
  • file.write("текст\\n") — записать строку (не забывать \\n).

Блок 5 · Условия и циклы

if, for, while True и break

Условия и циклы управляют логикой программы. На примерах с котами: можно вывести только голодных котов, повторять добавление новых котиков, пока пользователь хочет это делать, и т.п.

Условие if / elif / else

if cat.age < 1: print("Это котёнок") elif cat.age <= 7: print("Это взрослый кот") else: print("Это почтенный кот-пенсионер")

Цикл for по списку котов

for cat in cats: print(cat.name, "любит", cat.favorite_food)

Цикл while True для добавления “цитат” котиков

while True: answer = input("Добавить новую цитату котика? (да/нет) ") answer = answer.lower() if answer == "да": quote = input("Введите цитату: ") cat_name = input("Имя кота: ") with open("cats_quotes.txt", "a", encoding="utf-8") as file: file.write(quote + "\\n" + "(" + cat_name + ")" + "\\n") else: break
  • while True: — бесконечный цикл, пока не встретится break.
  • break — выход из цикла.
  • answer.lower() — приводим ответ к нижнему регистру, чтобы сравнение было надёжным.
while True + break if / elif / else for cat in cats
Блок 6 · Строки и типы данных

Строки, split(), int(), lower()

Почти всё, что приходит из input() и файлов — строки. Потом ты превращаешь их в числа или разбираешь на части.

Метод split() — разбить строку про кота

line = "Мурзик 3 тунец" data = line.split() # ["Мурзик", "3", "тунец"] name = data[0] age = int(data[1]) favorite_food = data[2]
  • split() без аргументов делит по пробелам.
  • data[0], data[1], data[2] — обращение к элементам списка по индексу.
  • int(data[1]) — превращает строку "3" в число 3.

Метод lower() — для ответов пользователя

answer = input("Кот доволен своей миской? (Да/Нет) ") answer = answer.lower() if answer == "да": print("Кот мурлычет :)") else: print("Кот недоволен и требует ещё корма!")

Основные типы данных

  • str — строки: "Мурзик", "тунец", "3".
  • int — целые числа: возраст кота — 3, количество мисок — 5.
  • float — числа с точкой: вес кота в кг — 4.5.
  • bool — логические значения: True / False, например cat_is_hungry = True.
  • list — список: ["Мурзик", "Снежок", "Луна"] или [cat1, cat2, cat3].
str → int: int("3") "Текст".lower() line.split()
Блок 7 · Дополнительно

Замер времени, списковые включения и словари

Несколько полезных приёмов, которые часто встречаются: измерение времени выполнения кода, списковые включения (короткий способ собрать список) и словари для хранения информации о котиках.

Модуль time — измеряем, сколько код “думал”

import time start_time = time.time() # например, вывод всех котиков print_cats(cats) end_time = time.time() print("Код работал:", end_time - start_time, "секунд")
  • import time — подключить модуль.
  • time.time() — текущее время в секундах.
  • Разница между end_time и start_time — время работы кода.

Списковые включения (list comprehensions)

# список возрастов всех котов ages = [cat.age for cat in cats] # список имён только взрослых котов (старше 2 лет) adult_cat_names = [cat.name for cat in cats if cat.age > 2]

Словари (dict) — информация о коте по ключам

cat_info = { "name": "Мурзик", "age": 3, "color": "серый", "favorite_food": "тунец" } print(cat_info["name"]) # Мурзик cat_info["weight"] = 4.5 # добавили новый ключ "weight"
import time [cat.age for cat in cats] {"key": "value"}

  • Добавить “цитату” кота → спросить текст и имя → открыть cats_quotes.txt в режиме "a" → write().
